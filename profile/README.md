## Overview
<img src="https://i.imgur.com/7dppjV4.png" alt="Web Desktop + AT Protocol" />

atproto OS represents an initiative to develop a web platform based on the Open Web Desktop framework, with deep integration of the AT Protocol. The primary goal is to provide specific modules for the AT Protocol, usable within a modular and adaptable interactive web environment offering a desktop experience.

### Your Digital Garden in a Windowed Environment on the AT Protocol

atproto OS aims to establish a controlled and customizable digital environment, leveraging the distributed and open architecture of the AT Protocol within an intuitive and familiar windowed user interface.

For a comprehensive understanding of the project, please refer to the main repository:  
[github.com/atproto-os/atproto-os](https://github.com/atproto-os/atproto-os)

## Focus and Objectives

* **Native Integration with the AT Protocol:** To deliver a seamless and comprehensive user experience for interacting with applications and data distributed on the AT Protocol.
* **Modular Web Desktop Environment:** To adopt a modular architecture inherited from Open Web Desktop, ensuring flexibility, customization, and extensibility.
* **Development of AT-Specific Modules:** To create dedicated software components for managing decentralized identifiers (DIDs), publishing and consuming records, and other core functionalities of the AT Protocol.
* **Coherent User Experience:** To offer a familiar and intuitive user interface, enabling users to interact with the AT Protocol through a traditional desktop paradigm.

### Next Steps and Vision

The atproto OS project operates as a division of the [Open Web Desktop](https://github.com/owdproject) organization, established to explore and showcase the potential of integrating web-based environments with the AT Protocol.
The core objective is to provide specialized modules for the AT Protocol, seamlessly operable within a modular and adaptable web environment simulating a desktop experience. The project is under active development.

We invite developers, researchers, and enthusiasts of the decentralized web to follow the project's evolution and contribute to its growth.
